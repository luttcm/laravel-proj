# Комплект документации по итоговому проекту

Проект: многопользовательская сетевая информационная система поддержки коммерческих расчетов, отчетности и корпоративной базы знаний.

- [Диаграмма классов](./class-diagram.md)
- [Проектная документация](./design-document.md)
- [Спецификации программных модулей](./module-specifications.md)
- [Сценарии тестирования](./test-scenarios.md)

Основание для описания:

- Laravel-приложение из `public_html/app`
- роли пользователей `admin`, `manager`, `finance`, `redactor`
- маршруты и модули, зафиксированные в `routes/web.php`
- модули авторизации, 2FA, расчетов, отчетности, новостей, справочников и базы знаний
